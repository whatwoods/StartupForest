# 创业森林官网 产品需求文档 (PRD) ## 一、产品概述 创业森林是一个面向创业者的活动平台，支持组织者发起创业活动，选手报名参赛、提交项目、参与投票，并对外展示历届活动成果。 ### 1.1 角色体系 - **访客**：浏览首页、活动列表、活动详情页。 - **选手**：报名、查询入选状态、提交项目、投票。 - **组织者**：创建和管理活动、审核报名、管理项目投票。 - **发起者**：系统最高权限，审核组织者申请，全局统计。 ### 1.2 活动生命周期 1. **报名中**：展示介绍 + 报名表单。 2. **入选公示**：隐藏报名入口 + 开放入选查询。 3. **活动进行中**：入选选手提交项目 + 投票。 4. **活动结束**：项目展示墙 + 获奖公示 + 活动回顾。 ## 二、页面清单 - **首页 (/)**：品牌介绍、活动入口、历届回顾。 - **活动列表页 (/events)**：筛选、卡片列表。 - **申请组织者 (/apply)**：提交申请表单。 - **活动详情页 (/event/:id)**：多阶段功能切换的核心页面。 - **用户中心 (/me)**：我的报名、项目、个人信息。 - **组织者后台 (/admin)**：活动管理、报名/项目管理、数据统计。 ...
